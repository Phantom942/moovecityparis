Google Search Console Verification

Ce fichier est utilisé pour la vérification de la propriété du site dans Google Search Console.

Pour vérifier votre site :

  1. Allez sur Google Search Console
  2. Ajoutez votre propriété (https://www.moovecity.fr)
  3. Choisissez la méthode de vérification "Fichier HTML"
  4. Téléchargez le fichier fourni par Google
  5. Remplacez ce fichier par le fichier de Google

Note : Ce fichier doit être remplacé par le fichier de vérification fourni par Google Search Console.
